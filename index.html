(LAMP) Laravel / MySQL
Project Name: Basic LAMP - Using laravel and MySQL through an ORM called Eloquent to create a blog site.
  1. console "git clone https://github.com/FreezeBerries/lsapp"
  2. console "composer install"
  3. console "npm install"
  4. console "cp .env.example .env" then update database creds
  5. console "php artisan key:generate"
  6. console "php artisan migrate"
  7. Start up apache and visit site. You can modify (apache/conf/extras/httpd-vhosts.conf) and (your host file on windows)
